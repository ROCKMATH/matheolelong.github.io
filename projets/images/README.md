# Dossier Images

Place ici les captures d'écran et images de tes projets.

## Conseils d'organisation

- Crée un sous-dossier par projet si tu as plusieurs images
- Nomme tes fichiers clairement : `nom-projet-screenshot1.jpg`
- Optimise tes images avant de les ajouter (compression, resize)
- Formats recommandés : JPEG pour photos, PNG pour captures d'écran avec texte, WebP pour meilleure compression
